What is Mojito?

  • Mojito is a Node.js module. It greatly benefits from the speed and scalability of Node.js.
  • Mojito is a model-view-controller (MVC) application framework. It allows developers to use a combination of configuration and an MVC architecture to create applications.
  • Mojito is built on YUI 3. The models and controllers in the Mojito MVC use Y.Base, and the addons are based on YUI Plugins
  • The Mojito command-line tool, besides being used to create and start applications, also offers developers with a variety of utilities.
